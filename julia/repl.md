# REPL stuff
   
### Clearall  & goto TOP:
```
Ctrl+L
```
   
## Package manager:

#### Enter package manager mode:
```
]
```
      
#### Status
Show list of package install    
```
status
```
    
#### add or remove package   
```add <package name>```    
```rm <package name>```   

#### Exit package manager
```<press backspace key>```
