# REPL stuff
   
### Clearall  & goto TOP:
```
Ctrl+L
```
   
## Package manager mode:

#### Enter package manager mode:
```
]
```
      
#### Status
Show list of package install    
```
status
```
    
#### add or remove package   
```add <package name>```    
```rm <package name>```   
   
## Help mode:

#### Enter Help modeL
```
?
```

## Exit any mode
```<press backspace key>```
